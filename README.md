# 📊 Logistics Operations Analytics Dashboard

An end-to-end Power BI project designed to analyze and monitor logistics operations. This dashboard provides actionable insights into delivery performance, resource utilization, and customer satisfaction to support data-driven decision-making.

---

## 📌 Problem Statement
Logistics operations often lack a centralized system to track key metrics such as total orders, delivery performance, and resource utilization. This project aims to solve that by building an interactive dashboard for better visibility and analysis.

---

## 🎯 Objective
- Track total orders and delivery trends  
- Monitor on-time delivery performance  
- Measure customer satisfaction (CSAT)  
- Analyze average delivery time  
- Evaluate hub, driver, and vehicle performance  

---

## 📂 Dataset Description
The dataset consists of 4 interconnected tables:

- **Orders:** Delivery details, time, delays, CSAT  
- **Drivers:** Driver information, experience, ratings  
- **Vehicles:** Vehicle usage, status, maintenance  
- **Hubs:** Hub details and capacity  

---

## 🧠 Data Modeling
- Orders ↔ Drivers (Driver ID)  
- Orders ↔ Vehicles (Vehicle ID)  
- Orders ↔ Hubs (Hub Name)  
- Orders ↔ Date Table (Date)  

---

## 📊 Key KPIs
- Total Orders  
- On-Time Delivery Rate (%)  
- Customer Satisfaction Score (CSAT %)  
- Average Delivery Time  

---

## 🚀 Features
- Delivery performance tracking  
- Hub capacity analysis  
- Driver performance insights  
- Fleet utilization monitoring  
- Month-over-Month (MoM) trend analysis
